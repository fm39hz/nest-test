# Nest API

[Nest](https://github.com/nestjs/nest) project for test .

## Installation

```bash
bun install --verbose
```

## Running the app

```bash
# development
bun run start --verbose

# watch mode
$ bun run start:dev --verbose

# production mode
$ bun run start:prod --verbose
```

## Test

```bash
# unit tests
$ bun run test --verbose

# e2e tests
$ bun run test:e2e --verbose

# test coverage
$ bun run test:cov --verbose
```
